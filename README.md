# HF Build Small Hackathon

This repository is a selected public snapshot of work from the Hugging Face
Build Small Hackathon.

## Collaboration

@cu is the human participant and product owner. Codex worked as the AI coding
agent and collaborator across product framing, implementation, packaging,
runtime checks, and evidence gathering.

@cu made the release decisions, chose what should be public, and reviewed the
submission scope. Codex helped write and modify code, organize artifacts, run
local checks, and keep the project workflow explicit.

`COAGENTS.md`, `TRIBAL.md`, and the workflow lane docs were key parts of that
collaboration system. They helped @cu and Codex split responsibilities, preserve
tribal knowledge, choose the right workflow lane for each task, and keep proof
claims honest while moving three hackathon ideas toward shippable demos.

## What This Repo Contains

This repo does not contain the full working repository. It contains only
selected files from active hackathon ideas, plus public-safe context and
evidence needed to understand the work.

Internal scratch work, secrets, local caches, private runbooks, raw agent
transcripts, and unreviewed traces are intentionally excluded.
